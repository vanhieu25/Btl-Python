## How to Contribute

### 1. Fork the repository
Click "Fork" button on the top right of this repository page.

### 2. Clone your fork
```bash
git clone https://github.com/YOUR-GITHUB-USERNAME/Btl-Python.git
cd Btl-Python
```

### 3. Add upstream remote
```bash
git remote add upstream https://github.com/vanhieu25/Btl-Python.git
```

### 4. Create a feature branch
```bash
git checkout -b feature/your-feature-name
```

### 5. Make your changes
- Follow the project structure in `docs/SPEC.md`
- Write code in `src/btl_python/`
- Add documentation in `docs/` if needed
- Write tests in `tests/`

### 6. Commit and push
```bash
git add .
git commit -m "feat: describe your changes"
git push origin feature/your-feature-name
```

### 7. Create Pull Request
Go to GitHub and create a Pull Request from your branch to `main`.

## Code Style Guidelines
- Use Vietnamese for comments and documentation
- Follow PEP 8 for Python code style
- Write meaningful commit messages
- Update documentation when adding new features

## Branch Naming Convention
- `feature/` - New features
- `fix/` - Bug fixes  
- `docs/` - Documentation updates
- `chore/` - Maintenance tasks
